Purpose

Fraud prevention and detection has become a major necessity for any web application. This Website provides the first line of defense for companies which collect user information for hosted events. Without this - there is no way to know if the end user is legitimate.

By using advanced learning algorithms in sklean, a model was built that trains on a captive data set with known fraud identified. Then by clicking predict, new data can be screened for fraudulant activity.
